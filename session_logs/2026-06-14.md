## 요약

## 주요 변경
- **플레이테스트 픽스 배치**: 튜토리얼 문구 평이화+속도+줄바꿈, 레벨업 lead-in 제거(어색),
  추천 멘트 상대비교+완성문장화, 보스 자폭 0뎀 barrier 버그(`Player.take_hit` 가드),
  물음표 방 VEIL 3줄 동시출력 버그(→await 순차), 보스/최종 클리어 후 레벨업 스킵(1+3).
- **글라이드 재설계**(`SkillTreeData`/`Player`): T1 활강 / T2 삼단점프 / T3 관통·추적. (T1에 활강+삼단
  점프 둘 다라 OP였음 → 분리.) 감시탑 깨진 게이트는 일반 XP로 강등.
- **상성 swap**: 저격수→방어막, 드론→글라이드. multishot은 일반 픽. 코드+문서 전체 동기화.
- **크레딧 정직화**(`Credits.gd`): Soomin Kim=Direction만, 제작=Claude/생성툴. `docs/contributions.md` 신규.
- **A 크레딧 메뉴**: 크레딧 끝 [다시 플레이하기]/[메인으로] 버튼. "다시 플레이하기" 포커스 시 글리치로
  "베일의 진실에 더 다가가기"로 모핑. `GameState.replaying` 신호 → 물음표 첫 단말기 변형(기기≠사람 회피).
- **PS#2 상시 VEIL 눈**: Stage HUD 우상단 작은 눈(BriefingVisual). 시야붕괴 시 눈·자막·마커 글리치
  (드롭아웃+지터=통신두절/EMP). `BriefingVisual` degraded 강화 + 실시간 반영, `_tick_subtitle_glitch`.
- **D 냉각 맵 리뉴얼**: `SteamVent.gd` 신규(주기 증기 분출 해저드). 냉각 맵 VERTICAL_UP→HORIZONTAL
  통로 + 증기 6 + 드론 3(상성 글라이드) + 고립 글라이드 게이트. 서사 훅("서버 식히는 곳…저도 어딘가").
- **스킬 반영 그래픽** 제안 backlog 등재.

## 결정 사항
- **다회차 인식 = 명시적 "다시 플레이하기"로만.** 기기 영속 visit_count는 부스에서 새 사람이 VEIL-1
  도입을 놓치게 함 → 엔딩 후 replay 버튼을 누른 경우에만 변형. 4번째 벽 정책과 일관.
- **글라이드 OP는 티어 분리로.** 삼단점프를 T2로 내려 T1 가치를 낮춤. 게이트는 T2(삼단점프) 전용.
- **냉각 = 가로 통로.** 증기 분출구(바닥 수직 제트)가 가로 통로에서 가장 읽기 좋고, 글라이드 게이트도
  새 레이아웃이라 진짜로 고립 가능. 드론 주력 → 상성 바뀐 글라이드 학습 맵.
- **크레딧은 정직하게.** 사람이 안 만든 영역에 개인명 안 넣음(사용자 지시). 보고서용 분담표 별도.

## 세션 2 — 루트 화면 정비 / 정찰병 / 타이틀 / 레버 / 자막

### 요약
플레이 피드백 추가분 처리. 전 맵 서사 훅 + 루트 카드 추천 스킬, risk/reward 감사, 감시탑 정찰병
내려쏘기, 타이틀 소제목, 외곽 레버 한 화면 + 냉각 레버, 데이터센터 자막 겹침, 냉각 doc 동기화.

### 주요 변경
- `RouteData`: 전 맵 description을 "존재 이유" 서사 훅으로(PALIMPSEST 테마). 블랙아웃 risk 2→3,
  핵심부 멘트 자연화, 비상 탈출로 최종 스테이지(6)에만 등장(미리 탈출 방지).
- `RouteMap`: 포커스 패널에 맵 소개 + 권장 스킬(상성, `_recommended_skill_line`) 표시.
- `Enemy`: 정찰병 `PATROL_FIRE_MAX_DY=48` — 높이 차 큰(등반 중) 표적엔 사격 안 함(감시탑 입구 내려쏘기 해소).
- `scenes/title.tscn`: 소제목 "5개의 임무"(7/5 불일치)→"침투 작전".
- `Stage`: ARENA(camera FIXED)에서 자막 상단으로(플레이어 겹침 해소). 외곽 레버 1300→2120(해치 옆),
  냉각 레버 비밀칸 추가(`_build_cooling_secret`, 레버 바로 위 해치).
- `world_layout`: 냉각 VERTICAL→HORIZONTAL 동기화.

### 결정 사항
- **risk/reward 표시 등급 = 클리어 보너스 XP**(인게임 오브와 별개). 냉각 게이트 XP3는 추가 보상이라
  reward 등급(이미 3)은 유지. 명확한 불일치는 블랙아웃(1-hit-fail인데 risk 2)뿐 → 3으로.
- **탈출로 = 항상 마지막.** 일반 모드 최종 스테이지(인덱스 6=7번째)에만. 선택 시 클리어=엔딩(보스 없이 빠진 결말).
- **정찰병 사격은 같은 높이에만.** 등반 중 내려쏘기는 불합리(점프 최고점에서만 반격) → 근접 돌진만 유지.
- **첫 레버는 비밀보다 명료함 우선.** 레버↔해치 한 화면으로 인과를 가르치고, 냉각에서 한 번 더 강화.

### 미해결 / 다음 할 일
- 플레이 확인: 정찰병 사격 체감, 탈출로 흐름, 추천 스킬 표시, 블랙아웃 위험, 레버 인과, 데이터센터 자막 위치.
- 정찰병 쿨다운 재사격 분기 dy 체크는 탭 매칭 이슈로 미적용(초기 교전 체크로 감시탑 입구는 해결됨 — 마이너).

렌더는 Claude가 창모드 godot을 직접 띄워 PNG를 Read로 확인하며 반복(사용자 --gen 불필요).

### 주요 변경
- **PosterCanvas**: 엔딩 행(A/B/C/D + FOUR ENDINGS·ONE MISSION / FOLLOW×FIGHT), 스샷 섹션 헤더,
  코너 브래킷 축소(58→40)·세로 레이아웃 전면 재배분, 타이틀 outline 9+크기 128(faux-bold).
- **ThumbCanvas(신규)** + **Poster**: --gen 시 4종 저장(기본 1240 / 2배 2480×3508 / 썸네일 150·300).
  2배는 별도 SubViewport + canvas.scale=2, 썸네일은 600 렌더 후 Lanczos 다운스케일.
- **스킬 그래픽**: Bullet(fire_boost 티어별 탄/잔상 + style_tier), CharacterArt(부착물 5종 +

### 결정 사항
- **포스터 = 100% 엔진 렌더.** 챗판 동등 콘텐츠 + 작가성 우위. 렌더는 Claude가 창모드로 직접.
- **렌더 검증 순서: 헤드리스 import(파싱) → 창모드(PNG).** 창모드 단독은 hang 위험.
- **20분 hang 원인**: 첫 창모드 렌더 직후 .gd 4개를 연달아 수정 → import가 소스 변경 감지로 재시작
  루프 추정. 코드 수정 완료 후 import는 exit 0(클린). 이후 렌더 중 파일 수정 금지.
- **스킬 부착물은 skills_changed 신호로 갱신**(add_skill emit → Player receive → attach 재실행).

## 세션 5 — 엔진 포스터 대개정 (사용자 피드백 반복)

### 주요 변경 (PosterCanvas.gd / ThumbCanvas.gd)

- 색 팔레트: 파랑 중복 제거 → 시안/코랄/앰버/민트 4색 구분.
- VEIL 대화창: 눈 아이콘 + 이름표 + "믿을수록 더 도와드릴 수 있어요"(게임 대화창 스타일).
- 카피 전면: 로그라인 강화, 인용구 교체, 섹션 라벨 한국어화(ONE MISSION 제거),

- 썸네일: 부제(surveillance roguelite) 제거 → 아이콘 + 제목만.

### 결정 사항
- 포스터 렌더는 Claude가 창모드로 직접 → PNG Read로 확인·반복. 사용자가 --gen 칠 필요 없음.
