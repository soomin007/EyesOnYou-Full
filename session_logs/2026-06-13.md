## 결정 사항
- **저격수 사거리는 '링', 포탑은 '선 길이 정합'으로 서로 다르게 처리.** 저격수는 투사체 없는
  히트스캔이라 위협이 *반경*(원) → 점선 링이 맞고, 포탑은 직선 탄이라 위협이 *선* → 선 길이를
  실제 사거리에 맞추는 게 맞다. 포탑은 선을 줄이지 않고 늘려 맞춤(watchtower 설계가 736px 가로탄
  도달을 전제로 하므로 총알을 줄이면 등반 압박이 깨짐).
- **포스터는 인엔진 + 실제 스크린샷.** 사용자 선택(인엔진 씬). AI 생성 일러스트는 게임 실제 화면과
  어긋나 "게임을 잘못 대변"하므로 배제. 키 비주얼은 BriefingVisual의 VEIL 눈(자산 의존 없는 _draw)
  재현 + Screenshotter로 찍은 실제 게임 화면.
- **스크린샷은 Stage를 자식 인스턴스화해 캡처.** change_scene는 하니스를 날려 캡처 불가 → 자식으로
  add_child 후 메인 뷰포트 캡처. GameState.record_route_choice로 라우트 세팅을 게임과 동일하게.
- **챗 버전 병행 지원.** 완전 일관성은 인엔진 PNG, 회화적 키 비주얼이 필요하면 챗 + brief 문서.

## 세션 3 — 어투 아크 재설계(ACT→신뢰+사망) / 난이도 분석 / trust 코드 이식 ①

### 요약
어투 아크의 트리거를 진행도(ACT)에서 **신뢰도+사망**으로 전환하는 큰 재설계를 확정했다. 설계 3종 문서를
쓰고, 게임 난이도를 코드 실측으로 심층 분석했으며, ★0 첫 단계인 `trust_score` 재작업을 코드에 이식했다.

### 주요 변경

  취약함 게이트(무사망 고수→COLD 가능), 튜토리얼 복선, 다회차 4번째벽 금지, 반영 순서.
- `docs/design/veil_pool_remap.md` (신규) — ACT1/2/3 풀=COLD/THAW/WARM 밴드 재키. 브리핑 2차원
  (내용=진행도 고정 / 어투=신뢰)·도달가능 밴드·grid 전수·사망 실력 오버레이.
- `docs/design/difficulty_analysis.md` (신규) — 코어 전투 실측, 4중 구조적 관대함, front-loaded+보스
  스파이크, 스토리=무실패, 난이도↔신뢰아크 상호작용.
- `scripts/GameState.gd` — trust 재설계 이식. 획득(추천 따라 클리어 +2/함께 고비 +2/독립 성공 +0),
  적립을 record_route_choice→on_stage_clear로 이동, `veil_trust_tier` 원값 재정의, `veil_register_band`
  (취약함 게이트), `veil_tone_color` 차가움→따뜻함 그라데이션, 게이지 0-시작 + `veil_trust_gauge_dots` 헬퍼.
- `scripts/EndingResolver.gd`+`Ending.gd`+`Settings.gd` — 엔딩 신뢰축=추천 수용률(≥50%)로 재튜닝.

- `docs/design/backlog.md` — ★0 등재(이식 6단계 + 완료 후 전체 md 최신화 약속).

### 결정 사항
- **어투=신뢰 / 친절=사망 두 축 분리.** 사망 딜레마(친절 vs 초보)는 축을 합쳐서 생긴 충돌 — 분리하면
  사망=안내강도↑(직관), 친밀은 신뢰로만. skilled는 "내가 필요없겠네" 물러섬 강조.
- **신뢰 변수=방안 B(통합) + 엔딩 분리.** trust_score는 0에서 climbing(register용), 엔딩은 추천
  수용률(도덕축)로 분리해 획득 인플레에 안 깨지게. 엔딩 커밋 분리로 revert 용이.
- **무사망→COLD 가능(취약함 게이트).** 따뜻함=관계라 한 번도 안 죽고 위기 안 겪으면 유대 미형성=COLD.
  반직관 아님(완벽한 도구 테마·skilled 물러섬과 운율). 독립적 성공 +0(이전 "성공시 조금↑" 철회).
- **대사는 작전 통신체로.** 교과서 격식 대신 "확인하겠습니다/띄워둘 테니/전방 유지하십시오"(핸들러 어투).
- **검증:** Godot 4.6 헤드리스 에디터 임포트 EXIT=0, 컴파일 에러 0(`--check-only`는 오토로드 미등록이라 부적합).
