## 주요 변경 (본편 EyesOnYou-Full)
- **저장소 분리**: 데모(EyesOnYou)에 `demo-v1` 태그 동결 → 새 repo `EyesOnYou-Full`(Public,
  fresh, github.io/EyesOnYou-Full) 시드·배포. deploy.yml은 repo 이름 의존성 없어 그대로 작동.
- `docs/design/playtest_round2.md` 신설: 2차 피드백 분석(짧음·더블점프 막힘·키배열·저격수/드론·
  보스2P) + 다회차 대사 적중 확인 + Phase A 반영표.
- `Player.gd`: coyote time(0.1s) + jump buffer(0.1s). `_handle_input(_delta)`→`(delta)` fix.
- 키 배열 개편(`project.godot`·`Settings.gd`·`Main.gd`): 사격 J·대시 K·스킬 L(Q 제거) +
  ZXC·화살표·마우스 보조(전부 동시), 마우스 1순위 강등. 화살표↑ 점프. SETTINGS_VERSION 3→4.
- 키 안내 동적화(`GameState.action_label`/`controls_hint_line`): InputMap 기반 단일 소스 →
  키 변경·리매핑 자동 반영. Tutorial/Stage 중복 힌트 제거.
- `MapData.gd` rooftops·watchtower 발판 재배치: 점프 등급 S(Δ≤95)/D(Δ~130), Δ160/180 제거.
  높이 압축(상승 2850→2150). watchtower 글라이드 게이트는 강등(일반 XP).
- `GameInfo.gd` 신설 + Title 패치노트 패널 + Briefing 로딩 팁.

## 결정 사항
- **데모 동결 정책 조정**: "콘텐츠(맵/적/스토리)는 동결, 조작/UX 버그픽스는 데모에도 백포트".
  데모 URL만 가진 응답자가 같은 불만(점프·키)을 계속 주는 걸 막기 위함. 발판 전면 재배치 같은
  대규모 작업은 본편 전용. 데모/본편이 fresh 히스토리라 cherry-pick 불가 → 파일 복사로 백포트.
- **키 동적화**: 하드코딩이 4곳+ 분산 + 리매핑 시 거짓 안내 → InputMap에서 라벨 동적 추출(단일 소스).
- **발판 재배치 우선**(사용자): 점프 자체가 아니라 발판 간격(Δ160 빠듯)이 "점프킹"의 원인.
  Δ130 여유로 통일하되, 1차에서 발판 추가로 등반이 길어져 2차에 높이 압축(25%↓).
- **글라이드 게이트 강등**: 압축 상층에서 인접 슬랩과 같은 높이라 글라이드 전용 격리 불가
  (rooftops 2026-06-12 전례). 제대로 된 게이트는 전용 맵(cooling)에서.
