# 2026-06-22 세션 로그

## 요약
인스타 홍보용 임팩트 스크린샷 생성 하니스(IgShotter)를 새로 만들고, "가만히 선 평범한
순간" 대신 교전/VEIL 경고/맵 개성/시스템 화면을 연출해 1920x1080 PNG 10장을 뽑았다.
포스터용 기존 Screenshotter는 건드리지 않고 별도 하니스로 분리.

## 주요 변경
- `scripts/IgShotter.gd` (+`.uid`), `scenes/ig_shotter.tscn` — 신규 캡처 하니스.
  - 1920x1080 borderless 창 + `content_scale_size=1280x720` 명시 고정 → 월드/HUD 1.5배 선명.
  - 적 직접 스폰(`Stage._spawn_enemy`)·웨이브 강제(`_spawn_wave`)·저격 조준 강제(`_start_aim`)·
    폭발/머즐 글로우 합성·카메라 줌/오프셋·자막 정리(`_purge_subtitles`) 헬퍼로 장면 연출.
- `poster_out/ig/*.png` — 산출물 10장(gitignore라 커밋 제외): combat_hero, combat_arena,
  veil_warning_sniper, map_rooftops/subway/datacenter/watchtower, route_fork, skilltree, boss_sentinel.
- `docs/design/known_issues.md` — 런타임 창 리사이즈 시 content_scale 미적용 함정 기록.

## 결정 사항
- 해상도: 사용자 확인대로 게임 원본 가로 화면비 그대로, 고해상(1920x1080). 인스타 4:5는 편집 단계.
  창 장식 손실/모니터 clamp을 피하려 borderless + 모니터 네이티브(1920x1080)로 캡처.
- SubViewport 대신 메인 뷰포트 사용: HUD(CanvasLayer, 고정 px)가 엔진 stretch를 못 받아 작아지는
  문제 때문. 메인 뷰포트+content_scale 고정이 월드+HUD를 함께 균일 확대해 가장 깨끗.
- 커버는 두 후보 제공 — combat_hero(클로즈업, 큰 캐릭터+폭발) / combat_arena(아레나 풀웨이브, 여러 명).
- 루트 분기는 스토리 stage1(폐쇄 지하철 위험2 vs 감시탑 위험3)으로 2갈래+위험 대비를 자연히 확보.
- 맵 컷은 진입/회피 자막이 겹쳐 어수선 → 캡처 직전 일괄 purge로 깨끗한 위협/정체성 컷.

## 세션 2 — 플레이 피드백 반영(흐릿함·벽 관통 조준·박진감)
사용자 지적 두 건 + 박진감 요청 반영해 하니스 보강:
- **흐릿함**: 프로젝트 `physics_interpolation=true`라 텔레포트한 바디가 물리틱 사이를 보간하며
  캡처 순간 잔상이 생김(HUD만 또렷했던 이유). → `root.physics_interpolation_mode = OFF`로
  매 프레임 정확한 물리 위치 렌더 → 캐릭터·탄 또렷.
- **벽 관통 조준**: `_force_aim`이 LoS를 무시하고 조준선을 강제로 그렸음 → `_aim_if_los`로 교체,
  실제 게임 `_has_line_of_sight`가 트인 저격수만 조준/발사(`_fire_at_player`로 실제 탄 streak).
- **박진감**: 교차사격(지붕 저격수 실발사+플레이어 부채꼴), 옥상 점프 사격, 같은-층(중층 랙)
  저격 교전+위 드론, 포탑 아래 응사 등 실제 메커니즘으로 재연출. datacenter는 플레이어를 중층 랙에
  올려 양쪽 저격수와 같은 높이→사선이 자연히 트이게(랙 위로 탄 통과).
- known_issues에 "런타임 리사이즈 content_scale 미적용" 함정 기록(세션1)에 더해 보간 함정은 본 로그로.

## 세션 3 — 플레이어 반투명 + 가짜 머즐 글로우 정리
- **반투명 원인 = 무적(invuln) 점멸.** `Player._update_visual`이 `invuln>0`이면 `visual.modulate.a`를
  0.4↔1.0로 흔든다(피격 표시). 하니스가 피격 방지로 `invuln=999`를 켜둬, 캡처 프레임이 0.4 위상에
  걸리면 플레이어가 반투명(combat_hero/veil는 우연히 1.0 위상). → 캡처 직전 `_freeze_for_capture`로
  적/적탄/플레이어를 정지 + `invuln=0` + `visual.modulate.a=1.0` 강제 → 모든 컷 불투명·또렷.
- **가짜 머즐 글로우 제거.** 실제 게임에 없는 별모양(`_spawn_muzzle_glow`)을 내가 덧댔던 것 — 전부 삭제.
  진짜 머즐 플래시(`Player._show_muzzle_flash`, `_try_attack`이 냄)만 남김. 둥근 폭발은 실제 Bomb
  blast와 동일해 유지(원하면 제거 가능).
