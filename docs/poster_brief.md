# Eyes on You — 포스터 제작 브리프

> 이 문서는 **Claude/GPT 챗에 그대로 붙여넣어** 게임 소개 포스터를 만들게 하기 위한 자료다.
> 인엔진으로 렌더한 버전은 `scenes/poster.tscn`(스크립트 `Poster.gd`/`PosterCanvas.gd`)이 만들고,
> 출력 PNG는 `poster_out/eyes_on_you_poster.png`에 있다. 스크린샷은 `poster_out/shots/`.

---

## 챗에 줄 프롬프트 (이 아래를 통째로 복사)

아래 정보를 바탕으로 인디 게임 **세로형(A4 비율) 소개 포스터**를 만들어줘. 톤은 "조용하고 서늘한 근미래
감시물" — 거의 검은 배경에 **시안(하늘색) 단색 강조** 하나만. 화려하지 않고 절제된, 다큐멘터리 같은 분위기.
한국어 포스터다. 첨부한 게임 스크린샷을 키 비주얼로 쓰고, AI로 새 일러스트를 지어내지는 말 것
(게임의 실제 화면과 어긋나면 안 됨).

### 게임 한 줄
**Eyes on You** — AI 파트너 'VEIL'과 함께 짧은 횡스크롤 스테이지를 클리어하는 로그라이트.
조언을 듣거나 무시한 선택이 쌓여, 마지막에 **VEIL이 누구였는지** 드러난다.

### 기본 정보
- 장르: 횡스크롤 액션 어드벤처 + 로그라이트
- 엔진: Godot 4.6 (GL Compatibility)
- 플레이 시간: 1회 8–15분
- 플랫폼: 브라우저(웹) + 데스크톱, Xbox 컨트롤러 지원
- 외부 의존성 없음(서버/계정/결제 없음)
- 플레이: soomin007.github.io/EyesOnYou

### 세계관 / 톤
근미래 민간 보안기업 **ARCTURUS DYNAMICS**의 시설 'SILO-7'. 플레이어는 현장 요원이고,
상황실 AI **VEIL**이 헤드셋 너머에서 위협을 짚어주고 길을 권한다. 핵심 정서는 **감시·신뢰·의존**.
VEIL은 **가끔 틀린다** — 그래서 늘 따르는 게 정답이 아니다. 차분하고 건조한 1인칭 작전 기록 느낌.

### 핵심 훅 — 4종 결말
두 축(VEIL 신뢰 / 전투 성향)이 직교해 결말이 갈린다:
- **A 완벽한 도구** — VEIL의 진짜 목적이 드러난다
- **B 혼자였던 사람** — VEIL은 의존받지 않기를 바랐다
- **C 공생** — 유일하게 VEIL에게 직접 묻는 선택지가 열린다
- **D 유령 임무** — 10초의 정적, 기록 없음

### 특징 (포스터에 넣을 4블록)
1. **VEIL — 당신을 보는 AI**: 화면 안팎의 위협을 미리 마킹·경고하는 상황실 파트너. 후반엔
   그 시야가 흐려지고 일부는 영영 안 보인다(시야 역전).
2. **5종의 적**: 정찰병(돌진)·저격수(조준선 노출 후 발사)·공습 드론(폭탄 투하)·자폭병(점멸 후 폭발)·
   방패병(정면 무효). 측면 둥지의 저격수는 글라이드로 덮친다.
3. **스킬 8라인 × 3티어, 3계열**: 전투(주황)·이동(하늘)·생존(초록). 레벨업마다 셋 중 하나 선택.
4. **12개 맵 · Dead Cells식 진행**: 스테이지마다 후보를 추첨. 위험(risk)을 감수할수록 보상(reward)이 큰다.

### 제목 & 카피
- 제목: **EYES ON YOU** (영문 그대로, 크게)
- 태그라인: **"누군가, 당신을 보고 있다."**
- VEIL 인용(분위기용): *"위험한 건 제가 먼저 볼게요. 화면 끝에 짚어둘게요." — VEIL*
- 정보 칩: 횡스크롤 로그라이트 · 8–15분 · 4종 결말 · Godot 4.6

### 비주얼 아이덴티티 (실제 게임 값)
- 배경: 거의 검정, 차가운 남색기 — `#0B0C10` 계열 (RGB 11,12,16)
- 강조색(유일): **VEIL 시안** — `#75DBFF` 계열 (RGB 118,219,255)
- 계열색(스킬): 전투 `#F89479` / 이동 `#8CD1F7` / 생존 `#94EBAE`
- 폰트: Pretendard (한글), 제목은 영문 산세리프 굵게
- 모티프: **감시 조리개 / 눈** — 동심원 링 + 회전 스캔선 + 가운데 시안 눈동자(레이더처럼). "Eyes on You"의 핵심 심볼.
- 질감: 아주 옅은 스캔라인, 네 귀퉁이의 가는 시안 코너 브래킷(감시 UI 프레임)

### 레이아웃 제안 (위→아래)
1. 상단 가는 키커: `ARCTURUS DYNAMICS · 현장 작전 기록 · OPERATION PALIMPSEST`
2. 감시 조리개(눈) 심볼 — 중앙 상단, 시안
3. 제목 EYES ON YOU + 태그라인 + 로그라인 2줄
4. 게임 스크린샷 3장 가로 스트립(시안 테두리 + 맵 이름 캡션)
5. 특징 4블록(2×2, 각 블록 앞에 계열색 사각형)
6. VEIL 인용 한 줄
7. 푸터: 플레이 URL + `Godot 4.6 · 개인 프로젝트 / 전시 데모`

## 참고 — 인엔진 버전과의 차이
챗 버전은 자유로운 그래픽/타이포가 가능한 대신 게임 실제 폰트·색과 미세하게 달라질 수 있다.
완전히 일관된 아이덴티티가 필요하면 인엔진 버전(`poster_out/eyes_on_you_poster.png`)을 쓰고,
더 회화적인 키 비주얼이 필요하면 챗 버전을 쓰되 위 색/모티프를 지키게 한다.
