## 2. 발견 — 주제별

### 2.1 저격수 배치 — 양호 (옛 우려 해소)
모든 sniper는 **고지대 발판/지붕/post**에 있고 아래로 LoS가 명확하다(rooftops·watchtower·cooling·subway).
평지 한가운데 방치 케이스는 **없음** → known_issues의 "글라이드 무의미" 함정은 현재 맵엔 해당 안 됨.
다만 **저격수를 "못 잡고 회피해야만 하는" 위치**(엄폐물 뒤·긴 갭 너머 단독 발판)는 아직 없음 — 지금은 모두
"올라가서 잡을 수 있는" 위치. backlog의 "회피 강제 저격" 아이디어는 여전히 미반영.

### 2.2 글라이드/대시 지형 — 이미 존재 (backlog B 부분 충족)
글라이드/대시가 유의미한 지형이 5개 맵에 있다: subway(천장 낮음), cooling(파이프 갭), blackout(좁은 발판),
rooftops·watchtower(저격 회피). **즉 "글라이드 유리한 맵이 없다"는 아님** — 있긴 하나 *강하게 유인되지 않음*이
실제 문제일 수 있음. 보강 방향: 이 5맵에서 글라이드/대시 보상(지름길·안전·아이템)을 더 또렷이.

### 2.3 발사 함정(총알 트랩) — 미구현 (명확한 빈칸)
위험요소가 **정적 가시(spike)·토글 가시뿐**. backlog B의 "지나가면 정해진 라인으로 발사하는 트랩"은 어디에도 없음.
→ 기동/회피 스킬 가치를 높이는 가장 직접적 신규 요소. 한 종(가로/세로 발사)부터 프로토타입할 만함.

### 2.4 드론(공중) — 후반 집중
drone은 cooling(2)·datacenter(W2 1)·blackout(2)에만. 첫 등장 stage 3~4(cooling). 공중 위협이 후반에 몰려
있어, 다중사격(matchup: drone→multishot) 학습 타이밍이 늦을 수 있음 — 중반 한 맵에 소수 드론 도입 검토.

### 2.5 난이도 곡선 — 대체로 매끈, 점검 포인트
- stage 0~1: back_alley(1/1)·rooftops(2/2) — 무난한 시작. ✓
- 중반: subway/sewers/cooling/ward(2/2~2/3) — 적 종류 늘고 함정 등장. ✓
- 후반: watchtower(3/3)·datacenter(3/3)·lab(보스). ✓
- **점검**: back_alley는 patrol 3마리·함정 0으로 매우 비어 있음(인트로라 의도적이나, stage1 재등장 시 단조로울 수 있음).
  escape도 patrol 2(의도적 숨 고르기). 두 맵은 "비어 있음"이 의도인지 재확인.

### 2.6 위험요소·연출 다양성 — 단조
함정이 가시 한 종류뿐. 토글 가시·드롭 발판·레버·해치는 있으나 "능동 위협"은 적+가시뿐.
다양화 후보: 발사 함정(2.3), 이동 위협(왕복 레이저/회전), 붕괴 발판, 추격 요소.

### 2.7 분기·보상 설계 — 양호
rooftops·sewers·cooling·watchtower 모두 "노출(XP)↔안전(HP)" + 비밀 통로의 3갈래로 잘 짜임. 유지 권장.

## 3. 폴리시 우선순위 제안

1. **발사 함정 1종 프로토타입** (가장 큰 빈칸·기동 가치↑) — 가로 또는 세로 라인 발사, 타이밍/대시 회피.
   적용 후보: subway(천장 낮은 긴 구간)·blackout(정밀 구간)·cooling(파이프).
2. **회피 강제 저격수 배치 1곳** — 엄폐물 뒤/긴 갭 너머 단독 발판에 sniper를 둬 "못 잡고 지나가야" 하게.
   기존 rooftops/watchtower 상층에 변형 추가가 저비용.
3. **글라이드/대시 유인 강화** — 2.2의 5개 맵에서 지름길·비밀·안전 루트를 글라이드/대시로만 닿게(보상 명시).
4. **중반 드론 소수 도입** — multishot 학습을 stage 2~3으로 앞당김(예: subway 또는 ward에 drone 1).
5. **back_alley/escape "비어 있음" 재검토** — 의도면 유지, 아니면 가벼운 함정/적 1개 추가.

> 4·5는 밸런스 영향이 있어 플레이테스트 동반 권장. 1·2는 신규 요소라 독립적으로 만들고 한 맵에서 검증 후 확산.
