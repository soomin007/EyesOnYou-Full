## 0. 총평

방향 채택. **"시야 = 신뢰"는 옳다.** 게임 제목이 *Eyes on You*이고 VEIL의 존재 이유가 "요원을 실시간으로 본다"인데 그 '봄'이 플레이에 비어 있었다 — v1은 그 빈 자산을 정확히 짚었고, 신뢰를 메뉴 산수에서 매 순간의 시야로 옮겨 진단 1·2·3·6·7을 한 수로 건드린다. §2-1도 안 깨고 오히려 "다섯 장치"를 시야 안으로 모았다.

다만 매력의 상당 부분이 **시야 큐라는 새 게임플레이 시스템**에 걸려 있고(§3), 그건 비싸고 "큐 따름" 판정 등 까다로운 설계가 남아 있다. 그래서 **텍스트만으로 지금 확보 가능한 것은 즉시 반영**하고, 시스템성 항목은 v2로 되돌려 함께 설계하기로 했다.

## B. v2로 되돌림 (설계가 더 필요 — 함께 정하자)

### B-1. 시야 큐 시스템 (v1 §3.1 / 3.2 / 3.3 대사 / 3.4 표시) — Tier 2, 새 게임플레이 시스템

이게 진단 1·2(메뉴 산수)의 *진짜* 해법이지만 실제 엔지니어링이다(`Stage.gd` 3000줄+, `Enemy.gd`, HUD, `GameState`). 코드를 보는 입장에서 **세 가지 설계 난점**을 먼저 풀어야 한다:

1. **"큐를 따랐다"를 어떻게 견고하게 판정하나?** "표시된 엄폐로 이동했나"를 시간·공간으로 추적해야 하는데, 잘못 잡히면 신뢰가 *임의로* 오르내려 오히려 진단 1을 악화시킨다. v1의 `trust += 작은 값`이 가린 핵심 난점. 판정 규칙을 구체화해줘 (예: 표시 후 N초 내 표시 지점 반경 진입 = 따름? 표시된 적을 건드리지 않고 통과 = 따름?).
2. **"적 처치 = 공격성"이 코어 루프와 충돌한다.** 이 게임은 *쏘는 게 주 동사*인 액션 플랫포머다. 매 교전을 도덕 판정으로 만들면 손맛을 짓누른다. 현재 설계가 aggression을 *경로 선택*에 둔 건 이 충돌을 피하려던 것이기도 하다. "전투를 벌하지 않으면서 측정"하는 안전선이 필요 — 예: "VEIL이 *명시적으로 피하라 표시한* 소수의 특정 적"으로 한정? 비살상 우회 *보너스*는 주되 일반 전투엔 페널티 없음?
3. **신규 인게임 시야 대사(§3.3)는 이 시스템 트리거에 종속**된다("왼쪽 위, 드론 둘. 보여요." 등은 VEIL이 무언가를 *표시*할 때 나오는 말). 시스템이 확정돼야 어디서 발화할지 정해진다. → 텍스트는 좋으니 시스템 확정 후 한 번에 반영.

### B-2. 신뢰 = 화면 밝기 (v1 §3.4 선택) — 보류 권장

**충돌 발견**: 지금 코드에 **접근성 "화면 밝기" 설정**(플레이어가 화면을 밝히는 슬라이더)을 막 추가했다. v1의 "저신뢰 → 화면 어두워짐"과 겹치면 (a) 저신뢰 플레이어가 밝기를 올려 연출을 무력화하거나 (b) 둘 다 어두워져 안 보이게 된다. v1 §7.2가 우려한 **부스/전시(10~20분, 신규 플레이어) 가독성 위험**이 이 접근성 기능과 겹쳐 더 커진다.
→ 신뢰의 시각화는 **밝기 대신 색(이미 tier별 청록↔붉은색) + 시야 표시 밀도/선행시간**으로 가는 게 안전해 보인다. 의견 줘.

### B-3. 엔딩 콜백 한 줄 (v1 §5.4) — 보류 / 의뢰인 판단

여백 우려 + 엔딩 본문 보존 원칙(§2-1). v1도 "강제하지 않음, A의 한 줄만 검토"라 했다. 시야-두려움 비트가 자리잡은 *뒤에* 그 콜백이 자연스러운지 판단하는 게 맞아서, A-3가 안착한 다음 v2에서 결정하자. (A: `"아까, 문 앞에서 제가 멈췄던 거. 기억나요?"` — 후보 보존.)

### B-4. 임계값·tier 경계 재튜닝 (v1 §3.1 밸런스 노트) — B-1 확정 후

시야 신호가 추가되면 한 판에서 누적되는 trust/aggression 총량이 바뀐다. **신호 총량이 정해진 뒤** 임계값 4·tier 경계를 다시 잡아야 한다. 지금 손대면 두 번 일한다. B-1과 묶음.

## C. Claude Code 추가 제안 (작가 검토용 — 미반영)

### C-1. 인트로에 시야 모티프 심기

A-3에서 S0 브리핑에 시야 모티프를 넣었는데, **첫 교신(인트로)**에도 한 줄 심으면 테마가 더 일찍 선다. 현재 인트로 마지막 줄 `외곽부터 가요. 차분히 살펴보면서.` 를 예컨대 `외곽부터 가요. 길은 제가 봐요.` 류로. 단 이건 v1에 없던 내 제안이라 **반영 안 함** — 채택 여부 알려줘.

### C-2. VEIL-1도 "본다" 동사로 묶기 검토

v1 §6이 "VEIL-1=임무만 봤다 / VEIL-2=요원만 봤다 / VEIL-3=둘 다"로 세 버전을 *본다*로 관통시키자고 했다. 좋은 방향. 현재 VEIL-1 대사는 강해서 유지 권장(v1 §5.3 동의)이지만, 혹시 한 줄을 시야 어휘로 살짝 물들일 여지가 있는지(예: "저는 임무만 봤어요." 같은) — 강한 원문을 해치지 않는 선에서만. 작가 판단.
