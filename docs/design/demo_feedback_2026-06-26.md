## 코드 대조로 확인한 사실
- **위협 화살표 각도**: 이미 `dir = (적 - 플레이어).normalized()` 동적 계산(VeilSight `_draw_edge_arrow`).
  "고정되어 어색"은 구 빌드/오해. 단 `EDGE_MARGIN=48`이라 가장자리에서 떨어진 건 사실(줄일 여지).
- **레버**: `LeverInteractable`은 점멸 빛(hint glow)만, **텍스트 프롬프트 0** → 피드백 유효.
- **점프 바인딩**: W·Z·Space·**Up** 모두 jump(현 코드). 코요테 타임/점프 버퍼 구현됨(`_coyote_t`/`_jump_buffer_t`).
- **엔딩 막판 선택**: 데모는 구 추천수용 2축. **본편(B2/B3)은 처리 선택 4종이 엔딩 결정** → 본편서 이미 해결.
