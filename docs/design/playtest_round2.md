# 플레이테스트 피드백 2차 분석 (2026-06-24)

> 데모(demo-v1) 공개 후 들어온 설문 응답 묶음. 총 16건(신규 ~13건이 06-24 집중).
> 이 분석은 본편 **Phase A**(분량·다양성, expansion_plan.md) 설계의 직접 입력이다.
> 1차 반영(보스 안내·드론/보스 크기·최종목적 노출)은 데모 demo-v1에 이미 들어가 있다.

## 전반 지표

- **재미 평균 ≈ 4.1 / 5** (분포 3~5). 핵심 루프·리플레이 훅은 성공.
- **난이도 분산**: 1=4건 / 2=4건 / 3=6건 / 5=1건 → 대체로 쉬움~적당(3이 적정 기준).
- **그래픽·효과음**: 거의 전원 "아주 만족". **BGM**만 상대적 약점(보통~"별로" 1건).
- 입력: 전원 키보드.

## 🔴 강한 신호 (우선순위 순)

### 1. 더블점프/플랫폼 올라가기 — 신규·다수·코드 원인 규명됨 ★최우선

여러 응답이 같은 증상을 보고:
- "더블점프 최대거리로 해야 플랫폼 올라가져서 답답(속도감 없음)"
- 아쉬운 점 = "더블점프" / "점프 타이밍 맞추는 것"
- **결정적 증언**: *"더블점프했는데 캐릭터가 플랫폼을 안 밟고 통과해 원상태로 돌아옴. 멈췄다가 점프+이동키 쓰면 올라가는데 딜레이도 있음"* (포탑+저격수 올라가는 수직 스테이지)

**코드 원인 (`Player.gd` `_try_jump`)**: **coyote time이 없다.**
- 가장자리에서 *걸어서 떨어지면* `is_on_floor()`가 즉시 false인데 `jumps_used`는 아직 0.
- 이때 점프 입력 → `is_on_floor()` false라 지상 풀점프(`JUMP_VELOCITY = -540`)가 아니라
  **공중 2단 분기(`-540 × 0.92 = -497`)로 약하게** 나간다. 그리고 더블점프 1회를 소모.
- 결과: "분명 점프했는데 높이가 안 나와 플랫폼에 못 닿음". 바닥에 완전히 멈췄다 뛰면
  (`is_on_floor()` true) 풀점프라 성공 → "멈췄다 하면 됨"과 정확히 일치.

**본편 수정안**:
1. **coyote time(~0.08~0.1s) 추가** — 가장자리 이탈 직후 짧은 윈도우 동안의 첫 점프는 지상 풀점프로 처리.
2. 제자리 더블점프(수평속도 0)의 정점이 일반 플랫폼 간격에 닿는지 레벨 단위로 검증
   (수평 입력 없이도 한 칸은 오를 수 있게 — 모서리 안착 실패 방지).
3. (선택) jump buffer(착지 직전 점프 입력 기억)도 같이 — 플랫포머 표준 2종 세트.

→ known_issues.md "게임 설계 함정"에 별도 기록(재발 방지).

### 2. 키 배열 불편 — J 사격

- "슈팅이 j인 게 불편" / "키 배열이 약간 일련되지 않아 불편" / "Q랑 J 쓰는 게 익숙하지 않음" (3건)
- **본편**: 사격을 마우스 좌클릭으로 일원화(이미 보강 입력 존재) + 키 리매핑 옵션 제공 검토.
  데모는 좌클릭 사격이 보강돼 있으나 키보드 안내(J)가 먼저 노출돼 키보드만 쓴 사람이 불편.

### 3. 저격수/드론 처치 난이도 (지속)

- "드론이 너무 죽이기 어려움 / 보스 아닌 몹이 너무 어려움"(난이도5 매긴 응답), "저격수 공격 어려움".
- 1차에서 드론·보스 크기를 키웠으나(demo-v1) 일부 잔존. 개인차 큼(난이도5는 1명).
- **본편**: 난이도 곡선 재설계(Phase A) 시 막1 잡몹 압박을 완만하게. 저격수 LoS/사선 노출 재점검.

### 4. 보스 2페이즈 밸런스

- "보스 2페이즈 속도 빨라지면 유도 없으면 총으론 사실상 못 때림" + (1차) "보스 사격법 모름".
- **본편**: 유도 미보유 빌드도 맞출 여지(텔레그래프·안전 윈도우 확대) — 막3 보스 재설계 시.

### 5. 분량 짧음 — 확증

- "너무 짧아서 아쉬움" **9/16건**. expansion_plan Phase A(분량 확대·막 구조)의 가장 강한 정당화.

## 🟢 긍정 / 적중 확인

- **다회차 VEIL 대사 적중**: *"크레딧 후 'VEIL 더 알아보기' 골랐더니 AI가 인사할 때 '익숙하다'는
  뉘앙스로 말하는 게 너무 좋았어요!!"* → 다회차 차별화(데모 세션 10~16 배선)가 정확히 먹혔다.
  **본편에서 이 방향(NG+/메타 진행)을 더 키울 근거.**
- 스토리/세계관 호평+갈증: "엔딩C 숨겨진 스토리에 놀람, AI가 진짜인지 궁금", 향후 희망 1위가 스토리 확장.
  → Phase B(스토리 깊이)의 수요 확인.
- 스킬트리 재미: "찍는 재미", "자유로운 전략", "레벨업이 매우 재밌음". → 스킬 확장(Phase A) 수요.

## 🟡 약한 신호

- 맵 선택 "뭘 골라야 할지 모름"(2건) — 화면 자체는 "쉬움"인데 선택지 *의미/결과*가 안 와닿음.
- "블랙아웃 발판을 밟지 말라는 게 무슨 뜻인지 모름" — 시야 붕괴 안내 문구 불명확.
- VEIL 자막/조작 안내 "헷갈림" 소수.
- BGM 약점 — 기존 인지(expansion_plan Phase D).
