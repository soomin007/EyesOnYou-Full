# 맵 세계 구조 + 특수 방 명세

> 11맵의 세계 형태(4 템플릿) + 보스/이스터에그/도전 방/웨이브 메커닉 정의.
> Stage.gd + MapData.gd가 이 문서를 따라 동작한다.

---

## 결정 요약

| 항목 | 결정 |
|------|------|
| 3.1 세계 형태 | **B안** — 4템플릿 (HORIZONTAL / VERTICAL_UP / VERTICAL_DOWN / ARENA) |
| 3.2 특수 방 | **기존 11맵 전환 + 숨은 트리거** — 핵심부=보스, 격리병동 내 이스터에그 트리거 |
| 3.3 보스 메커닉 | **기존 적 조합 + 3페이즈** — 신규 적 타입 없음 |
| 3.4 도전 방 룰 | **30초 + 노 데미지** — 실패 페널티 없음, 클리어 보상 XP +5 |
| 3.5 진입 조건 | 보스/도전=항상 보임, 이스터에그=격리 병동 레버+발판 트리거(숨김) |
| 3.6 부스 환경 | 이스터에그 1분, 도전 방 2분, 보스 3분 이내 |

## 4. MapData.gd 신규 필드

```gdscript
# 각 맵 딕셔너리에 추가
{
  "world_type":    "HORIZONTAL",       # 템플릿 식별자
  "world_size":    Vector2(4400, 720),
  "player_start":  Vector2(140, 540),
  "goal_type":     "POSITION",         # POSITION / ENEMY_CLEAR / SEQUENCE
  "goal_pos":      Vector2(4320, 540), # goal_type이 POSITION일 때만
  "camera_mode":   "HORIZONTAL",       # HORIZONTAL / VERTICAL / FIXED
}
```

### world_type → camera_mode 매핑
| world_type | camera_mode | 설명 |
|-----------|------------|------|
| HORIZONTAL | HORIZONTAL | 기존 가로 follow |
| VERTICAL_UP | VERTICAL | y축 follow, 위로 |
| VERTICAL_DOWN | VERTICAL | y축 follow, 아래로 |
| ARENA | FIXED | 전체 고정 |
