# Eyes on You — 난이도 심층 분석 (Claude 판단)

> 코드 실측 기반(2026-06-13). 추정 아님 — 모든 수치는 `Player.gd`/`Enemy.gd`/`BossSentinel.gd`/
> `GameState.gd`/`ExpOrb.gd` 인용. 목적: "이 게임이 누구에게 얼마나 어려운가"를 정량 판단.

---

## 0. 결론 먼저 (TL;DR)

- **대상은 부스 walk-up 캐주얼 + 접근성 사용자.** 난이도는 "숙련 게이머"가 아니라 이 기준으로 봐야 함.
- **스토리 모드 = 사실상 무실패**(피격이 HP를 안 깎음). 순수 체험 모드.
- **노멀 모드 = 중간 난이도, front-loaded(앞이 더 어렵고 뒤로 쉬워짐) + 막판 보스 스파이크.**
- **4중 구조적 관대함**(스테이지 체크포인트 · 매 스테이지 풀회복 · 상시 대시 무적 · 적응형 추천)이
  영구 벽을 원천 차단. 노멀의 난이도는 *실패*가 아니라 *재시도 횟수/좌절감*으로 나타난다.
- **가장 위험한 두 지점:** ① 무스킬 상태의 첫 전투(상대적으로 제일 셈), ② 노멀 보스(24HP·3페이즈).

## 1. 코어 전투 살상력 (실측)

| 항목 | 값 | 출처 |
|---|---|---|
| 플레이어 기본 HP | **3** | GameState `player_max_hp=3` |
| 모든 적 공격 데미지 | **1** (정찰/저격/드론/폭격/방패/보스근접 전부) | Enemy/Boss |
| 피격 후 무적 | **0.8s** (hp T2면 1.0s) | Player `INVULN_AFTER_HIT` |
| 대시 무적 | 0.18s, 쿨다운 0.7s, **상시 보유**(베이스라인) | Player `DASH_*` |
| 기본 사격 데미지 | 1 (fire_boost T1이면 2) | Player `_spawn_bullet` |
| 적 HP | 저격/드론/폭격 **1**, 정찰 **2**, 방패 **3** | Enemy |
| 적 텔레그래프 | 저격 조준 0.7s · 정찰 돌진충전 0.45s · 폭격 무장 0.7s | Enemy |

**판단:** "3 HP / 1뎀 / 0.8s 무적" = **회복창이 있는 3대**. 한 번 맞아도 0.8s 동안은 연타로 안 죽는다.
상시 대시 무적(0.18s)이 **패닉 버튼**이라 텔레그래프만 읽으면 회피 가능. 텔레그래프 0.45~0.7s는
논게이머도 반응 가능한 길이. → **코어는 "낮은 HP지만 읽기 쉽고 회피 수단 있음"** = 빡세 보이지만 관대.

## 2. 구조적 관대함 — 영구 벽을 막는 4중 안전망

1. **스테이지 = 체크포인트.** 죽으면 그 스테이지만 재시도(`death_count++`). **런 전체 손실 0.**
2. **매 스테이지 진입 시 HP 풀 회복**(`on_stage_clear` 주석). → 누적 소모전이 아니라 *스테이지별 생존*.
   한 스테이지만 넘기면 다음은 새 출발.
3. **위험도 플레이어 선택 + VEIL 적응형 추천.** `competence_tier`(최근 피격·죽음)가 *고전 중*이면
   쉬운 맵을 추천(`RouteData`). 난이도가 부분적으로 자기선택 + 가이드됨.
4. **회피/생존 스킬 안전망:** 비상부활(1~2회), 방어막(주기적 1회 무효), hp(+1~2 HP), 대시강화 무적연장.

→ **결론: 노멀에서도 플레이어를 영구히 막을 수 없다.** 난이도는 "벽"이 아니라 "보스를 몇 번 재시도하나"
정도로 나타난다. 부스 환경(짧은 1회 플레이)에 적합한 설계.

---

## 3. 난이도 곡선 — front-loaded + 보스 스파이크

**스킬 스노볼:** 적 처치 1XP(ExpOrb VALUE=1) + 스테이지 클리어 1~3XP, 레벨당 8XP →
대략 **스테이지당 ~1레벨** → 7스테이지에 6~8개 스킬 획득 → 2~3개 라인이 T2~T3.
fire_boost T1만 떠도 사격이 2뎀 = 대부분 적 원킬, hp 2티어면 5 HP. **뒤로 갈수록 확연히 강해진다.**

```
체감 난이도(노멀)
높음 │█                                   █  ← 보스(24HP·3페이즈)
     │█                                  ██
중간 │██                            ███████
     │ ███                    ███████
낮음 │   ██████████████████████
     └──────────────────────────────────────
       s0   s1   s2   s3   s4   s5   s6(보스)
       ↑무스킬 첫 전투(상대적 최고난도)
```

- **가장 어려운 순간 = 시작(무스킬·3HP)과 끝(보스).** 중반은 스킬이 붙어 가장 쉬움.
- 이 모양은 이상적 램프(점증)와 약간 어긋남 — **초반 바운스 위험**(아래 §5).

---

## 4. 스토리 모드 = 사실상 무실패 (정밀)

- `GameState.damage_player`가 `if story_mode: return` → **HP가 절대 안 깎임** → `is_dead()` 영영 false.
  **전투로는 죽지 않는다.**
- 추가 완화: **드론 제외**(낙하 패턴 인지 어려움), **보스 P1만**(HP 24→8, P2/P3·미사일·소환 없음).
- 유일한 실패 경로 = **블랙아웃 도전방**("한 대만 맞아도 끝", 시간제) — 그러나 이건 **opt-in 챌린지**라
  일반 진행이 아님.
- **판단:** 스토리 모드는 *긴장이 거의 없는 순수 체험 모드*. 논게이머·접근성 사용자가 스토리를 끝까지
  보게 하는 게 목적이라 **의도된 난이도 바닥**. 단점: "이겼다"는 성취감이 약함(아래 §6 권고).

## 5. 위험 지점 상세

### ① 무스킬 첫 전투 (가장 큰 부스 이탈 위험)
walk-up 플레이어의 *생애 첫 싸움*이 자기 파워 대비 제일 셈(스킬 0·3HP·낯선 조작). 여기서 2~3번 죽으면
부스 관람객은 그냥 떠난다. 완화책은 있음(VEIL이 쉬운 외곽 추천, 풀 회복) — 하지만 **이 지점이 곡선상
가장 약한 고리.**

### ② 노멀 보스 (난이도 정점)
24 HP·3페이즈, P3는 폭탄 0.9s·미사일 2.5s·잔당 소환으로 가장 공격적. 캐주얼에겐 *긴 체력바 + 압박*이
벽처럼 느껴질 수 있다. 완충: 페이즈 전환 시 1.2s 무적·정지, 자폭 버퍼(HP≤5 무적 자폭). 스토리는 통째 우회.
(2026-06-23 피드백 반영: 폭탄 간격 전 페이즈 완화 P1 1.5→1.7 / P2 1.0→1.2 / P3 0.7→0.9, 보스·드론
크기·피격박스 확대, 진입 시 전투 안내 자막 — §7 권고2 참조.)

## 6. 난이도 ↔ 신뢰 아크 상호작용 (이번 세션 작업과 직결)

방금 설계한 trust+death 어투 아크가 **난이도 적응 레이어**로 자연스럽게 맞물린다:

- `death_count` → `competence_tier` → **적응형 맵 추천 + (신규) 안내 강도**. 고전할수록 쉬운 길 + 강한 안내.
- **취약함 게이트(무사망→COLD):** 가장 *쉬운* 플레이(한 번도 안 죽음)가 가장 *차가운* VEIL을 낳는다.
  → 따뜻함의 감정 보상이 **"조금 고전하는 중앙값 플레이어"에게 최적화**된다. 부스 다수가 정확히 그 층 →
  **정렬이 좋다.** 고수는 "내가 필요 없겠네"(난이도·내러티브 둘 다 "불필요") 일관된 결을 받는다.

즉 난이도 곡선과 감정 곡선이 *같은 신호(death/competence)*로 구동돼 서로를 보강한다.

## 7. 권고 (선택 — 현행도 부스엔 충분)

1. **초반 바운스 완충(우선순위 높음):** 첫 죽음 시 VEIL이 쉬운 경로를 *더 강하게* 강조(이미 부분 존재 —
   §6 struggling 안내강도와 통합). stage 0 적 수를 risk 0.8보다 더 낮춰도 됨(첫 1분 보호).
2. **노멀 보스 완화 (2026-06-23 피드백 반영 — 적용됨):** "보스 사격법 모름·맞으면서 쏘나" 혼란 →
   ⓐ 폭탄 간격 전 페이즈 완화(P1 1.5→1.7 / P2 1.0→1.2 / P3 0.7→0.9 — 위 권고와 정합),
   ⓑ 보스·드론 시각+피격박스 확대(보스 ×2.0→2.5·56×40→70×50, 드론 시각 1.3배·32×24→42×32)로
   "보이는 만큼 맞음", ⓒ 진입 시 1회성 전투 안내 자막. 잔여 모니터링: 보스 재시도 횟수 추이.
3. **스토리 성취감(선택):** 무실패라 "이겼다"가 약함. 접근성 우선이면 현행 유지가 정답. 원하면 *약한*
   실패(넉백/짧은 리스폰, HP는 유지)나 "긴장 토글"을 옵션으로 — 단 기본값은 무실패 유지 권장.

## 8. 한 줄 요약

> **노멀은 "빡세 보이지만 구조적으로 관대한" 중간 난이도(앞이 어렵고 보스가 정점), 스토리는 무실패 체험
> 모드.** 부스/접근성 대상에 잘 맞고, 유일한 실질 리스크는 *무스킬 첫 전투의 초반 이탈*이다.
